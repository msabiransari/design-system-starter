## Remaining for v1.0

| Item | Priority | Notes |
|------|----------|-------|
| **Real production pilot** | Critical | One actual app in production using the system. Fixtures prove CSS; a real app proves adoption. |
| **Visual regression baselines committed** | High | 64 baseline screenshots exist locally. Must be committed to git and CI must fail on unexpected diffs. |
| **Angular 7 fixture completed** | Medium | Skeleton exists. Needs full Angular 7 app + old PrimeNG + Node 12 Docker/CI job. |
| **CHANGELOG.md** | Low | Formal release notes for 0.2.0 and onwards. |

## Post-v1.0

| Item | Notes |
|------|-------|
| Migration codemod | Auto-rewrite hardcoded colors to tokens. Risky for legacy apps; manual shim is safer for v1. |
| More adapters | Bootstrap, Ant Design, etc. |
| Figma Tokens export | JSON mirror for design teams. |
| React/Vue primitive wrappers | Framework-specific component libraries on top of tokens. |

## Release Checklist for v1.0.0

- [ ] All CI checks pass (lint, contrast, build, e2e).
- [ ] Visual regression baselines committed and stable.
- [ ] At least one production app using the system.
- [ ] Angular 7 fixture validated (or explicitly deferred with documented risk).
- [ ] CHANGELOG.md updated.
- [ ] Version bumped to 1.0.0 in `package.json`.
- [ ] Git tag `v1.0.0` created.
- [ ] npm package published.
